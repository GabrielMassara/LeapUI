# LeapUI

LeapUI é uma extensão para navegador que permite selecionar e manipular elementos de páginas web de forma interativa. Com uma interface intuitiva, você pode editar, duplicar e mover elementos diretamente na página.

## Como funciona

A extensão adiciona uma camada de interação sobre qualquer página web. Ao ativar o LeapUI, você pode:

**Seleção de elementos**: Passe o mouse sobre qualquer elemento da página para destacá-lo. As informações do elemento aparecem em tempo real na barra superior.

**Menu de contexto**: Clique em qualquer elemento para abrir um menu com opções de ação:
- Copiar seletor CSS do elemento
- Editar o texto do elemento diretamente na página
- Duplicar o elemento

**Modo movimento**: Ative o modo movimento para reposicionar elementos usando as teclas direcionais do teclado.

## Atalhos de teclado

- **Alt + I**: Ativar/desativar a extensão
- **Escape**: Desativar a extensão ou sair do modo atual
- **Ctrl + D**: Duplicar elemento selecionado
- **Enter**: Selecionar elemento para movimento
- **Setas direcionais**: Mover elemento no modo movimento
